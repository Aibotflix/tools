URL Encoder & Decoder

Encode URLs to percent-encoded format or decode them back. Full UTF-8 support for international characters and emoji.

What is URL encoding?

You need URL encoding when putting user input into a URL - search queries, form data, API parameters. Special characters like spaces, &, =, and ? have reserved meanings in URLs and must be encoded to avoid breaking the structure.

What is the difference between encoding a URL and encoding a URI component?

This tool correctly encodes non-ASCII characters (emoji, accented letters, CJK characters) as UTF-8 byte sequences before percent-encoding. Many naive encoders mangle these characters.
