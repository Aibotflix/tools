What does CSS minification do?

It strips out everything the browser doesn't need: whitespace, line breaks, comments, trailing semicolons. The CSS still works exactly the same but the file is smaller and your site loads faster.

Will minification break my CSS?

Nope. It just strips whitespace, comments and trailing semicolons. No selectors get renamed, no rules get merged, no values changed. What comes out works exactly like what went in.
