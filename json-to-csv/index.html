JSON to CSV Converter

Turn a JSON array into a CSV spreadsheet. Perfect for exporting API data, database results, or any structured data.

How do I use this?

Paste a JSON array into the left box. That's basically it — the CSV shows up on the right as you type. Got nested fields like {"address": {"city": "Paris"}}? Those flatten into address.city columns automatically. Hit Copy CSV or Download .csv when you're done.

What data is supported?

Pretty much anything you throw at it. Arrays of objects, a single object, arrays of arrays — they all work. Strings, numbers, booleans, nulls, nested objects, even arrays inside objects (those get turned into JSON strings inside the cell). If you can JSON.stringify it, this tool can probably convert it.

Is my data private?

Yes. Your JSON never leaves your browser. The conversion happens locally — nothing hits a server.
