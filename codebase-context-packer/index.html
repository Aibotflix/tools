Codebase Context Packer

Drop a folder, get a Markdown blob with all file contents - ready to paste into ChatGPT, Claude, or any LLM prompt.

What file types are included?
    
Text files  -  source code, Markdown, JSON, YAML, configs. Images, PDFs, compiled binaries and other non-text files are skipped.
    
Is my code uploaded?
    
No. All processing happens in your browser. The output goes to your clipboard or a file you download.
    
How big a folder can I use?
    
Limited by whatever your browser can handle in memory. For big codebases, pick only the folders you need.
