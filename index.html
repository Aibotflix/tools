FAQ

Are these tools free?

Yes. Every tool is 100% free with no usage limits and no account required.

Do you store the data I enter?

No. The tools run entirely in your browser — the code executes on your device and nothing you type is sent to any server.

How is this site supported?

The site is currently free with no ads. We may add unobtrusive advertising later to keep it running. Hosting is free on GitHub Pages.
