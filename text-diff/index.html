Text Diff Checker

Got two versions of a file and need to know what's different? Paste them side by side. Green is new, red is gone.

Paste text in both boxes above to see the diff.

How to read a diff

Lines highlighted in green with a + are new. Red with a - got removed. Plain lines stayed the same. The line numbers on the left show you where things sit in the original vs the new version. Good for comparing code changes, spotting edits in a draft, or double-checking a config file before deploy.

Is my data private?

Yep. Your two texts never leave your browser. The diff runs locally - nothing gets sent to any server.
