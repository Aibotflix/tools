What is binary?

Binary is a base-2 number system using only 0 and 1. It is how computers store and process data at the hardware level. Octal (base 8) is used in Unix file permissions (chmod 755). Decimal (base 10) is what humans use daily. Hexadecimal (base 16) uses 0-9 and A-F, common in memory addresses, color codes (#FF5733), and debugging.

What is hexadecimal used for?

Hexadecimal (base-16) is used to represent binary data compactly. Each hex digit represents 4 bits. It is common in memory addresses, color codes (#FF5733), and debugging.
