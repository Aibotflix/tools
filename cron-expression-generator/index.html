What is a cron expression?

A cron expression is a string of 5 fields — minute, hour, day of month, month, day of week. Star (*) means every. For example, 0 9 * * 1-5 means 9:00 AM on weekdays.

How do I run something every 5 minutes?

Use */5 in the minute field: */5 * * * *. The slash means start at 0, then every 5 minutes.

Does this support a seconds field?

No. This tool uses standard 5-field cron — minute, hour, day-of-month, month, day-of-week. Some systems like Quartz and systemd support an optional leading seconds field, but this generator doesn't.
