What is Protocol Buffers?

  
Protocol Buffers (protobuf) is a binary serialization format developed by Google. It's compact and fast, used in gRPC, config files, network protocols, and more. Unlike JSON, it's not self-describing — you need a schema to fully interpret the data.

  
What input format does this accept?

  
Paste hex-encoded data (with or without spaces/0x prefix) or base64-encoded data. The tool auto-detects the format. You can grab hex dumps from Wireshark, network logs, or binary file exports.

  
What do the wire types mean?

  
Wire type 0 is a varint (variable-length integer). Wire type 1 is a 64-bit fixed value. Wire type 2 is length-delimited (strings, bytes, embedded messages). Wire type 5 is a 32-bit fixed value. Types 3 and 4 are deprecated group markers.
