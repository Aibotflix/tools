HMAC Generator

Generate HMAC-SHA256 and HMAC-SHA512 signatures from a message and secret key. Uses the Web Crypto API.

—

What is an HMAC?

HMAC (Hash-based Message Authentication Code) is a signature that proves a message was sent by someone who knows the secret key. It combines a hash function (like SHA-256) with a secret key to produce a unique signature for each message.

When would I use HMAC?

HMAC is used to verify API request authenticity, validate webhook payloads, sign JWT tokens, and implement challenge-response authentication. If you need to prove "I know the secret" without sending the secret, use HMAC.

Is the key visible to anyone?

No. The key and message are processed entirely in your browser using the Web Crypto API. Nothing is sent to any server.
