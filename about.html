About

Boring Tools is a small collection of free, no-nonsense online utilities that run entirely in your browser. No accounts, no tracking, no firehose of pop-ups — just tools that do one thing well.

Everything here is open and built to last: simple HTML, CSS, and JavaScript hosted on GitHub Pages, so it costs effectively nothing to run and won't disappear behind a paywall.

Have an idea for a tool, or found a bug? Open an issue on GitHub.
