What is Markdown?

Markdown is a lightweight markup language created by John Gruber. You write plain text with simple syntax (# for headings, ** for bold, - for lists) and it renders as formatted HTML. It is used in READMEs, documentation, forums, and note-taking apps.

Does this editor save my Markdown?

No. Your text stays in the browser and is never saved to a server. You can copy the rendered HTML or download your Markdown as a .md file.
