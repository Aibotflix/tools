What is a hash?

The Web Crypto API (crypto.subtle.digest) computes SHA hashes natively in the browser. MD5 is implemented in pure JavaScript since browsers don't ship a native MD5 API.

Is MD5 secure?

MD5 has known collision attacks. Use it for non-security checksums only. For password hashing or digital signatures, use SHA-256 or SHA-512 with a proper KDF (like PBKDF2 or bcrypt).

Which hash algorithm should I use?

For password storage, use SHA-256 or SHA-512 with salt. For file checksums, SHA-256 is the modern standard. MD5 is fast but cryptographically broken — use it only for non-security checksums. SHA-1 is deprecated for security use.
