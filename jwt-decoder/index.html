What is a JWT?

JSON Web Token (JWT) is a compact, URL-safe token format used for authentication. It has three parts: header (algorithm), payload (claims), and signature.

Is it safe to paste my JWT here?

Yes. Decoding happens in your browser — the token is never sent to a server. JWTs are encoded, not encrypted, so decoding just makes the readable content visible.

Can this tool verify a JWT signature?

No. Verifying a signature needs the secret key or public certificate. This tool just decodes the header and payload so you can read them.
